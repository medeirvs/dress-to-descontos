Ficou com alguma dúvida?

Aqui vão algumas informações para te ajudar a economizar

1. Como utilizar os cupons de desconto Dress To?

Após copiar o cupom, vá até o carrinho de compras no site da Dress To e cole o código no campo de cupom. Clique em “OK” e veja o desconto ser ativado.

2. Os cupons são cumulativos com outras promoções?

Nem sempre. A maioria dos descontos não são cumulativos com outras promoções, mas isso depende das regras de cada oferta. Sempre verifique os detalhes antes de finalizar o pedido.

3. Posso usar dois cupons juntos?

Sim! O código de vendedora MX8M pode ser utilizado junto com o cupom de desconto, o que permite acumular benefícios em uma única compra.

4. O cupom é válido para todos os produtos?

A maioria dos cupons se aplica a produtos selecionados.
